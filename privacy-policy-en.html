Privacy Policy

Last updated: December 2025

1. Overview

Gloomy Console (the "App") is a mobile JavaScript console application provided by GloomyStore (the "Developer").

Gloomy Console does not collect any personal information from users.

2. Information We Collect

This app does NOT collect the following information:

  • Personal identification information (name, email, phone number, etc.)
  • Location data
  • Device identifiers
  • Usage statistics or analytics data
  • Code written by users

3. Data Storage

All data generated in the app (code snippets, settings, etc.) is stored locally on your device only.

  • Saved code snippets
  • Editor settings (theme, font size, etc.)
  • Execution history

This data is never transmitted to external servers. When you delete the app, all data is removed from your device.

4. Network Usage

This app operates offline by default.

Network connection may be required when:

  • You execute code that calls external URLs

The app itself does not transmit any data externally.

5. Third-Party Services

This app does not use advertising, analytics tools, or any other third-party services.

6. Children's Privacy

This app does not collect personal information from children under 13. Since the app does not collect any personal information, users of all ages can safely use it.

7. Changes to Privacy Policy

This privacy policy may be updated. Any changes will be posted on this page.
